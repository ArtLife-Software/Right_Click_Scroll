## 🚀 主要功能

這款工具能讓您的普通滑鼠瞬間進化，透過簡單的右鍵移動，實現高階滑鼠才有的全方位捲動體驗。

* **全方位捲動**：以「右鍵平移」代替實體滾輪，支援垂直與**水平**雙向捲動。
* **高階功能平民化**：無需購買昂貴的高階滑鼠，即可擁有橫向捲動功能。
* **雙重方案**：
    * **替代方案**：實體滾輪損壞時的完美救星。
    * **加強方案**：實體滾輪完好時，提供更直覺、更大範圍的操控感。

---

## 🎮 操作方式

只需簡單的滑鼠組合動作，即可精準控制頁面：

1.  **基本捲動**：在目標視窗上 **按住右鍵** 並 **上下或左右移動**，即可觸發對應方向的捲動。
2.  **渦輪加速**：在右鍵移動的同時 **按下左鍵**，捲動速度將提升為 **2 倍速**。
